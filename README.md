# Regatta Tracking

## Description

Some jupyter notebooks I wrote to help reduce tracking data from sailing.  Mostly from Wednesday Night Races on the Pontchartrain.  Each notebook is kinda my thought process and the questions I want to answer based on the specifics of the race, so it may seem a little all over the place.

The project uses MatPlotLib, PyProj, Pandas, and GeoPandas.  To process wind and tracking data.

Unless otherwise state the tracking data was captured by a legacy Garmin Quatix watch.  Tracks were clipped using OpenCPN to remove all the pre and post race sailing.

I enjoy sailing (I know I don't seem like the type but its a fun technical team sport and participation is free so try it).  More specifically, I enjoy sailing other peoples boats, and I enjoy reducing GPS data...  Now I can do both!
